One-time setup needed for live AIS positions

aisstream.io blocks direct browser connections for security reasons.
You need a free Cloudflare Worker proxy (takes ~3 minutes, free forever).

  1. Go to dash.cloudflare.com → Workers & Pages → Create → Create Worker
  2. Delete all the default code and paste in the contents of worker.js (included in the download)
  3. Click Deploy — note the worker URL (e.g. sulmara-ais.yourname.workers.dev)
  4. Go to Settings → Variables → add variable AIS_KEY = your aisstream.io API key → Save & Deploy
  5. Paste your worker URL below and click Connect
